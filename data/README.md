# Dataset access

The original AIS dataset used in the associated manuscript is not redistributed in this repository due to licensing and data ownership restrictions.

Please obtain the Ushant Traffic Separation Scheme AIS dataset from the original data provider.

## Expected input format

Trajectory files should be named:

```text
traj_1.txt
traj_2.txt
...
```

Each file should be semicolon-separated and contain at least:

```text
x;y
```

Optional columns:

```text
vx;vy;t
```
